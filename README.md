# 프로젝트 소개

**스페이스 클라우드 사이트를 참고한 사용자간의 다양한 공간 공유  플랫폼 사이트**
- 사용자는 호스트와 게스트의 역할을 가질 수 있으며 호스트는 공간을 등록하고 게스트는 호스트가 등록한 공간을 예약 할 수 있다
<br><br>
